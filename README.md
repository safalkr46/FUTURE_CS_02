## Project Overview

Phishing attacks are one of the most common cybersecurity threats used to steal sensitive information such as passwords, banking credentials, and confidential organizational data.

This project demonstrates how phishing emails can be analyzed using:

- Email content inspection
- URL and domain analysis
- Email header investigation
- Risk classification
- Cybersecurity awareness techniques

---

## Objectives

- Identify phishing indicators in suspicious emails
- Analyze spoofed sender domains and malicious URLs
- Perform email header analysis
- Classify emails based on risk severity
- Create awareness and prevention documentation
- Improve phishing detection and reporting skills

## Key Findings

The analysis identified several phishing indicators commonly used in real-world phishing campaigns, including:

- Spoofed sender domains
- Urgent and fear-based language
- Suspicious verification links
- Credential harvesting attempts
- Generic greetings
- Social engineering techniques
- Business Email Compromise attempts
